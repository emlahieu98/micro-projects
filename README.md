## Getting Started

Getting up and running is as easy as 1, 2, 3.

1. Technology
    - NestJs(NodeJs Framework)
    - MySQL2
    - TypeOrm
2. Install your dependencies

    ```
    cd path/to/
    npm install
    ```

3. Start your app

    ```
    add .env same .env.example
    Run docker-compose
    docker-compose up -d
    ```
